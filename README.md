# Пример
Создаем экземпляр класса
```
 $validator = new \Ddis\Meta\Validator();
 ```
 Устанавливаем строку для проверки
 ```
 $validator->setString("[]");
 ```
 Запускаем проверку
```
 $validator->validate();
 ```
 ## Ответ
 **True** - если все правильно 
 
 **False** - если нет
 
 **Error** - если возникла ошибка
